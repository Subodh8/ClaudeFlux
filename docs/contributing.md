# Contributing

See [CONTRIBUTING.md](../CONTRIBUTING.md) in the project root for full guidelines.

## Quick Start

```bash
git clone https://github.com/Subodh8/ClaudeFlux.git
cd ClaudeFlux
./scripts/dev-setup.sh
```

## Areas for Contribution

- **Core runtime** — Worker execution, DAG scheduling, budget enforcement
- **Dashboard** — React components, visualizations, real-time updates
- **Examples** — New workflow templates for common use cases
- **Documentation** — Guides, tutorials, API reference
- **Testing** — Unit tests, integration tests, benchmarks
